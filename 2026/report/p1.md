**“나만의 C++ 계산기 클래스 만들기”**

### 과제 목표

* **C++ 기본 입출력 + 클래스 + 캡슐화(private/public)** 를 직접 써서 “C++답게” 프로그램을 만들어 본다.
* 1강에서 배운 **struct vs class 차이**, **Set() / Add() 구조**를 스스로 재현한다.

---

## 과제 요구사항

### 1) 프로그램 이름

* `MyCalculator.cpp`

### 2) 기능 요구

아래 기능을 **클래스**로 구현하세요.

#### (A) 클래스 설계

* 클래스 이름: `MyCalculator`
* 멤버 변수: `a`, `b` (정수, **private**)
* 멤버 함수(최소):

  * `void Set(int x, int y)` : a, b 저장
  * `int Add()` : a+b
  * `int Sub()` : a-b
  * `int Mul()` : a*b

#### (B) main()에서 동작

1. 사용자에게 `a b` 두 정수를 입력받는다.
2. `MyCalculator` 객체를 만들고 `Set(a,b)` 호출
3. 다음과 같이 결과를 출력한다.

예)

```
두 정수를 입력하세요: 10 20
Add: 30
Sub: -10
Mul: 200
```
