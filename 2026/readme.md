아래는 **주당 4시간(이론 2h + 실습 2h)** 기준으로, **쉽고·재미있고·유용한** 방향의 **대학 표준 C++ 15주 강의 목차(8주 중간, 15주 기말)** 예시입니다.
(도구는 Visual Studio / CMake+VSCode 중 택1로 진행 가능)

## 1~15주 주차별: 이론 제목 + 실습 제목

| 주차  | 이론(2h)                                   | 실습(2h)                                 |
| --- | ---------------------------------------- | -------------------------------------- |
| 1주  | C++ 맛보기: “C와 뭐가 다르지?” + 개발환경 세팅          | Hello C++ + 입출력 + 간단 게임(숫자 맞히기)        |
| 2주  | 변수/타입/연산 + 문자열(string) 기초                | 콘솔 유틸 만들기: 단위변환기(온도/길이/환율 가짜데이터)       |
| 3주  | 조건문·반복문 + 디버깅(브레이크포인트, Watch)            | 미니게임: 가위바위보/랜덤 이벤트 + 로그 출력             |
| 4주  | 함수, 참조(reference), 기본 매개변수               | “기능 분리” 리팩토링: 메뉴형 콘솔 앱                 |
| 5주  | 배열·vector + 범위기반 for + 알고리즘 맛보기          | 성적관리기(추가/삭제/정렬/통계) with vector         |
| 6주  | 포인터는 “필요할 때만”: new/delete 개념 + 안전한 대안    | 동적 데이터 다루기 + 포인터 디버깅 실습(왜 위험한지 체감)     |
| 7주  | 객체지향 1: 클래스/객체/캡슐화/생성자                   | OOP로 바꾸기: “학생/과목/성적” 클래스로 재구성          |
| 8주  | **중간시험**                                 | **중간시험(실기 포함 가능)**                     |
| 9주  | 객체지향 2: 상속/다형성/가상함수(필요할 때만)              | 플러그인 구조 흉내: “도형(Shape) 그리기(텍스트)”       |
| 10주 | 파일 I/O + 예외처리 + 간단 포맷(JSON/CSV 개념)       | 저장/불러오기: CSV로 성적관리기 데이터 영속화            |
| 11주 | 표준 라이브러리(STL) 핵심: map/unordered_map, set | “검색이 빠른” 연락처/사전 앱 만들기                  |
| 12주 | 모던 C++ 핵심: auto, range, const, 스마트포인터    | 메모리 누수 0 만들기: unique_ptr/shared_ptr 적용 |
| 13주 | 템플릿/람다 기초 + 재사용 가능한 코드 사고                | 범용 함수 만들기: 통계/필터/정렬을 템플릿+람다로           |
| 14주 | 성능/구조화: 빌드(CMake), 모듈화, 간단 테스트           | 팀 종합과제 마무리: 구조 점검 + 데모 리허설             |
| 15주 | **기말시험**                                 | **기말시험(프로젝트 발표로 대체 가능)**               |

## 개별과제 5개 (제목)

1. **“나만의 콘솔 미니게임” (랜덤/반복/함수 활용)**
2. **“성적관리기 v1” (vector로 CRUD + 통계)**
3. **“파일 저장/불러오기 v1” (CSV로 내 데이터 영속화)**
4. **“연락처/사전 앱” (map 또는 unordered_map 기반 검색)**
5. **“메모리 안전 리팩토링” (raw pointer → 스마트포인터 전환 보고서 포함)**
