### 기본 입출력과 프로그램 구조

**○ 기본 입출력 (Input / Output)**

* C++에서는 표준 라이브러리 **`iostream`**을 사용하여 입력과 출력을 수행한다.
* **`cout`**은 화면에 데이터를 출력할 때 사용한다.
* **`cin`**은 키보드로부터 데이터를 입력받을 때 사용한다.
* 출력 연산자 **`<<`**, 입력 연산자 **`>>`**를 사용하여 데이터를 처리한다.

예시

```cpp
#include <iostream>
using namespace std;

int main() {
    int a;
    cout << "숫자를 입력하세요: ";
    cin >> a;
    cout << "입력한 숫자: " << a << endl;
}
```

---

**○ 프로그램 구조 (Program Structure)**

* C++ 프로그램은 보통 다음과 같은 구조로 이루어진다.

1. **전처리기 지시문**

   * `#include` 등을 사용하여 필요한 라이브러리를 포함한다.

2. **네임스페이스 선언**

   * `using namespace std;`를 사용하여 표준 라이브러리를 쉽게 사용할 수 있다.

3. **main 함수**

   * 프로그램의 시작 지점이며 실행의 중심이 되는 함수이다.

4. **실행 코드**

   * 변수 선언, 연산, 입출력, 함수 호출 등이 포함된다.

5. **return 문**

   * 프로그램이 정상적으로 종료되었음을 운영체제에 알린다.

예시 구조

```cpp
#include <iostream>
using namespace std;

int main() {
    // 실행 코드
    cout << "Hello C++" << endl;
    return 0;
}
```

✔ 한 줄 요약
**C++ 프로그램은 `main()` 함수에서 시작하며, `cin`과 `cout`을 사용해 데이터를 입력하고 출력한다.**
